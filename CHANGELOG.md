## v0.2 (2019-09-03)

First release, features:

- bip39 助记词,bip44 多币种私钥推导
- BTC 
    - new transaction
    - sign transaction
    - create multisig address
- Omni (same as BTC)
- ETH
    - new transaction
    - sign transaction
    - simple multisig support (deploy contract, contract call/transaction)
    - ERC20 support
